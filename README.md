# LINE Bot เช็กรอบรถ

บอทนี้ตอบคำถามว่า “รถจะถึงจุดขึ้นกี่โมง” โดยให้ลูกค้ากดเลือก **วันที่ → จุดขึ้น → จุดลง → รอบรถ** แล้วคำนวณจากเวลาออกรอบที่สถานีต้นทางอัตโนมัติ

เวอร์ชันนี้ยังไม่รับจองและไม่รับเงิน: ตอนจบจะแนะนำให้ลูกค้าติดต่อแอดมินเพื่อจอง/โอนตามระบบเดิม

## ข้อมูลที่แอดมินแก้เองได้

### `data/routes.json`

สร้างแยก 1 รายการต่อ 1 ทิศทาง เช่น `RY-KOR` และ `KOR-RY` เพราะป้ายกับเวลาของทั้งสองขาไม่จำเป็นต้องเหมือนกัน

`minutesFromOrigin` คือจำนวนนาทีที่รถใช้จากต้นทางถึงป้ายนั้น ตัวอย่าง `กม.10` มีค่า `40` หมายถึง รถที่ออกระยอง 06:00 จะถึง กม.10 เวลา 06:40

> ข้อมูลที่ใส่มาเป็นตัวอย่างจากตารางที่คุยกัน ต้องตรวจสอบและแก้ชื่อป้าย/นาทีให้ตรงกับเส้นทางจริงก่อนเปิดใช้งาน

### `data/schedules.json`

เพิ่มรอบรถได้ 2 แบบ

```json
{ "id": "everyday-0600", "date": null, "routeId": "RY-KOR", "departureTime": "06:00", "active": true }
```

- `date: null` = ใช้เป็นตารางปกติทุกวัน
- `date: "2026-07-22"` = ใช้เฉพาะวันนั้น และจะแทนตารางปกติของเส้นทางนั้นทั้งวัน
- `active: false` = ปิดรอบชั่วคราว

## วิธีเปิดใช้งาน

1. สร้าง **Messaging API channel** ใน [LINE Developers](https://developers.line.biz/console/)
2. คัดลอก `.env.example` เป็น `.env` แล้วใส่ Channel access token และ Channel secret
3. ติดตั้งและรัน

```bash
npm install
npm start
```

4. นำเว็บเซิร์ฟเวอร์ขึ้นออนไลน์ที่มี HTTPS แล้วกำหนด Webhook URL ใน LINE Developers เป็น

```text
https://ชื่อโดเมนของคุณ/webhook
```

5. เปิด `Use webhook` และปิด `Auto-reply messages` ใน LINE Official Account Manager เพื่อไม่ให้ข้อความซ้ำกัน

เมื่อลูกค้าเพิ่มเพื่อนหรือพิมพ์ข้อความ บอทจะเริ่มขั้นตอนเช็กรอบรถทันที
